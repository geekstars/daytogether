# How long have you been together? 365 days or 730 days or longer?

Day Together is a love day counting app with visual gentle interface and heart drop effect.

# Outstanding features of Day Together - Love days count

- Show the number of days together, love each other

- Update avatar and name of both on the software interface

- Synthetic wallpaper wallpapers as background for you to unleash your choice

- Save the memories of the two of you

The application is still in the process of completion, so we hope to receive everyone's suggestions and contributions.
